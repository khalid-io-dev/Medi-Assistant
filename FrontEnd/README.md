# ✅ README.md – Frontend (React + Tailwind)

# 🏥 CliniQ – Frontend Interface Clinique

## 📌 Description

CliniQ Frontend est l’interface utilisateur permettant aux professionnels de santé d’interagir rapidement avec l’assistant médical intelligent.

L’application offre :

- Une interface moderne et intuitive
- Authentification sécurisée
- Interaction en temps réel avec l’API RAG
- Dashboard administrateur
- Historique des requêtes médicales

---

# 🚀 Fonctionnalités

## 👤 Authentification

- Login / Register
- Gestion JWT
- Routes protégées

## 💬 Assistant Médical

- Interface de chat fluide
- Affichage des réponses contextualisées
- Historique personnel des requêtes

## 📊 Dashboard Administrateur

- Gestion des utilisateurs
- Historique global
- Statistiques d’usage

---

## 🗂️ Structure du projet

```bash
src/
 ├── components/      # Navbar, routes protégées
 ├── contexts/        # AuthContext
 ├── layouts/         # Admin, User, Auth layouts
 ├── pages/           # Chat, Login, Dashboard
 ├── services/        # Connexion API
 ├── App.jsx
 └── main.jsx
```

---

## 🛠️ Technologies

* React
* Vite
* Tailwind CSS
* Axios
* React Router
* Docker

--- 

## ⚙️ Installation

### 1️⃣ Cloner

```bash
git clone https://github.com/your-username/cliniq-frontend.git
cd cliniq-frontend
```
### 2️⃣ Installer

```bash
npm install
```

### 3️⃣ Lancer

```bash
npm run dev
```
